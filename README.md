## Tecnologías

- React
- TypeScript
- TailwindCSS
- Vite JS

## Instalación

1 Clonar el repo

```bash
 git clone https://github.com/Joaquin-Bianchi/wallbit-challenge.git
```

2. Instalar las dependencias

```bash
  npm i
```

3. Correr el proyecto

```bash
  npm run dev
```
