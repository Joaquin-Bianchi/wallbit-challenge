# Wallbit Frontend Challenge

Challenge de carrito de compras para el stream de Goncy x Wallbit

## Tecnologías

- React
- Typescript
- TailwindCSS

## Instalación

1 Clonar el repo

2. Instalar las dependencias

```bash
  npm i
```

3. Correr el proyecto

```bash
  npm run dev
```
